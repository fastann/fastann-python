# FastANN Python Binding


## Install
thanks for [setuptools-rust](https://github.com/PyO3/setuptools-rust)

### prerequirement

```
pip install -r setuptools-rust setuptools wheel
```

### release
```
python3 setup.py install --user
```

### debug
```
python3 setup.py develop --user
```
